# InstaCore
Instagram Bot written in C# using Selenium WebDriver

# Auth
 - Login
 - Proxy
# Profile
 - Get Followings Count
 - Get All Followings
 - Get Following by position
# Posts
 - Enter Post
 - Comment Post
 
Only tested with Chrome Selenium WebDriver
 
## Technologies

* C#
* .Net Core 3.1
* Seleniun Module
